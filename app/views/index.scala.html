Player Endpoints

GET /player/

Retrieve All Players


        [{
            id: 422,
            firstName: "Bob",
            lastName: "Dylan",
            email: "bob.dylan@@website.com"
        }, {
            ...
        }]

GET /player/{id: Int}

Retrieve the player with the given id


        {
            id: 422,
            firstName: "Bob",
            lastName: "Dylan",
            email: "bob.dylan@@website.com"
        }

POST /player/ [json]

Create the player with the given data


        {
            firstName: "bob",
            lastName: "dylan",
            email: "test@@test.com"
        }
        

        {
            success: true
        }

DELETE /player/{id: Int}

Delete the player with the given id


        {
            success: true
        }

Rating Endpoints

GET /eloRating/?playerId=2&leagueId=4&gameTypeId=5

Retrieve All GameTypes

Each query param is optional. No params will return all ratings, and each available param will reduce the set by that player/leage/gameType. Combinations are allowed


        [{
            eloRating: 1155,
            playerId: 66,
            leagueId: 1,
            gameTypeId: 6,
        }, {
            ...
        }]

POST /gameResult/ [json]

Post the result of a game, which will update EloRatings


        {
            player1Id: 52,
            player2Id: 21,
            didPlayer1Win: false,
            leagueId: 4,
            gameTypeId: 6
        }
        

        {
            success: true
        }
